This is a template project that can be used to easily start working with ebitengine.

It utilizes the following libraries:
 * Ebitengine - https://github.com/hajimehoshi/ebiten
    - Core graphics library
 * EbitenUI - https://github.com/ebitenui/ebitenui
    - UI Library
 * ResourceManager - https://github.com/bird-mtn-dev/resource-manager
    - Resource Management - Fonts, images, etc
 * StageHand - https://github.com/joelschutz/stagehand
    - Scene Management
 * Goeasy18n - https://github.com/eduardolat/goeasyi18n
    - Translations
 * Ebitengine-Input - https://github.com/quasilyte/ebitengine-input
    - Input management
 * Kamera - https://github.com/setanarut/kamera
    - Camera management
